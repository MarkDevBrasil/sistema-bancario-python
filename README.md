# 🏦 Sistema Bancário em Python

Projeto desenvolvido em Python com o objetivo de praticar:

- Funções
- Estruturas de repetição
- Condicionais
- Match Case
- Variáveis
- Simulação de sistema de login
- Operações bancárias

## Funcionalidades

### Sistema de Registro
- Cadastro de CPF
- Cadastro de senha

### Sistema de Login
- Autenticação de usuário
- Verificação de CPF e senha

### Banco
- Consultar saldo
- Depositar dinheiro
- Sacar dinheiro
- Sair do sistema

## Tecnologias

- Python 3

## Como executar

Clone o repositório:

```bash
git clone https://github.com/seuusuario/sistema-bancario-python.git
```

Entre na pasta:

```bash
cd sistema-bancario-python
```

Execute:

```bash
python sistema-bancario-python.py
```
